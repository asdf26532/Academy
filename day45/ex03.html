DOM

Document Object Moel

  • 문서 객체 모델
  • 페이지 내의 요소를 JS로 처리하게 만들어둔 객체

DOM으로 태그 탐색 방법

종류 설명
getElementById ID 선택자로 요소 접근
getElementsByClassName Class 선택자로 요소 접근
getElementsByTagName 태그 선택자로 요소 접근
getElementsByName name 속성 값으로 요소 접근
querySelector 지정 선택자로 요소 접근 (한개만 접근 가능)
querySelectorAll 지정 선택자로 요소 접근 (여러개 접근 가능)
  • ID 선택자
  • Class 선택자
  • Class 선택자
  • Name 속성
  • Tag 선택자
