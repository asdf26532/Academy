HTML
- Hyper Text Markup Language
- 다른 페이지에 링크가 걸린 문서
- Web에서 표준으로 널리 사용되는 문서
- HTML는 tag라고 하는 <>로 이루어져 있다
- tag의 종류는 다양하고 web browser가 해석한다
- tag는 open 태그와 close 태그로 존재한다
- close가 없는 경우도 있음
- tag 사이에 작성된 내용을 element라고 하고 tag의 효과를 받는다

Web
- HTTP 프로토콜로 제공되는 서비스를 의미
- 수많은 HTML 문서가 서로 연결되어 있고 이러한 페이지에서 정보를 주고 받는 서비스
