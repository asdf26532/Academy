이벤트(event)

  • 웹 브라우저에서 발생하는 어떠한 행위
  • JS는 이러한 이벤트를 처리하기 위한 언어
1. 마우스 이벤트
속성 설명
click 요소를 클릭시 발생하는 이벤트
dbclick 요소를 더블 클릭시 발생
mousedown 요소를 누르는 동안
mouseover 요소 위로 마우스가 옮겨질 때
mouseout 요소 밖으로 마우스가 벗어날 때
2. 키보드 이벤트
속성 설명
keypress 키보드 키를 눌렀을 때
keydown 키를 계속 누르는 동안
keyup 누르던 키를 땔 때
3. 문서 이벤트
속성 설명
abort 페이지가 완전히 로딩되기 전 취소 했을 때
error 페이지가 제대로 로딩되지 않은 경우
load 페이지 로딩이 완전히 끝난 경우
resize 페이지 화면 크기가 변경시
scroll 페이지가 스크롤될 때
unload 페이지를 벗어날 대
4. 폼 이벤트
속성 설명
blur 폼 요소에서 커서가 빠져 나올 때
change 체크 상태가 변경된 경우
focus 폼 요소에 커서가 들어왔을 때
reset 폼이 다시 시작될 때
submit 폼 데이터가 제출될 때
