sementic 태그

의미를 가지는 태그

  • 직접적인 효과가 있는 태그는 아님
  • 웹 브라우저 및 검색 엔진이 찾기 좋은 태그
  • HTML5에서 부터 지원하고 이전에는 div만 사용했다

종류

  1. header : 페이지 상단부 (로고, 검색 엔진 등)
  2. nav : 네비게이션. 페이지 메뉴 구성
  3. main : 주 컨텐츠
  4. section : 주 컨텐츠를 구별하는 영역
  5. article : 각 컨텐츠 마다 사용되는 영역
  6. aside : 좌/우측 사이드 메뉴
  7. footer : 페에지 하단부 (상호명, 저작권 고지 등)
