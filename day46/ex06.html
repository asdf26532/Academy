BOM

Brower Object Model

  1. Window : 브라우저를 조작하는 객체. 최상위 객체
  2. History : 방문 기록 정보를 조작하는 객체
  3. Location : URL 정보를 조작하는 객체
  4. Document : DOM, 문서를 조작하는 객체

Window 객체

  • innerHeight : 문서 높이
  • innerWidth : 문서 너비
  • outerHeight : 브라우저 높이
  • outerWidth : 브라우저 너비
  • moveTo(x, y) : 브라우저를 지정 좌표로 이동
  • open(url, target, option) : 새 브라우저(= 팝업) 오픈
  • close() : 브라우저를 종료
  • resize(x, y) : 지정 크기로 변경
